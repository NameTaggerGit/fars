# FARS — Технологический стек (кратко)

## Frontend
- **React 18** + **TypeScript**
- **Vite** — сборка
- **TailwindCSS** — стили
- **Zustand** — состояние
- **TanStack Query** — запросы к API
- **Framer Motion** — анимации
- **Socket.io-client** — real-time
- **React Router v6** — роутинг
- **WebRTC** (simple-peer / native) — звонки

## Backend
- **NestJS** + **TypeScript**
- **Prisma** + **PostgreSQL**
- **Redis** — сессии, кэш, pub/sub для Socket.io
- **JWT** + **Refresh tokens**
- **Socket.io** — WebSocket
- **Multer** + **S3/MinIO** — файлы
- **Nodemailer** — email
- **class-validator** — валидация DTO

## Инфра
- **Docker** + **docker-compose**
- **env** — конфиг
